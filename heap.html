Heap Sort

                Heap sort works by converting the array into a binary heap, which is a tree-based data structure. 
                The root element is swapped with the last element in the array and the last element is removed 
                from the heap. Heapify the elements in the  array and repeat until a max or min heap is obtained.

                Worst-case time complexity: O(nlogn)
